New Product Setup Checklist

This theme uses a custom product flow (size drawer, sticky bar, desktop layout). Follow these steps before publishing a new product.

1) Choose the product template

  • Use the template product.new-product (type new-product; file: templates/product.new-product.json).
  • Legacy layouts: product (main-product) and product.cheyenne-product exist but do not carry the size drawer/sticky bar experience.

2) Variants: sizes (required)

  • Add an option named exactly Size or Tamanho on the product.
  • Even one-size items need a single Size value so the size drawer can render and submit.
  • Keep inventory tracking on variants to power availability and “low stock” states.

3) Variants: colors (for swatches)

  • Add an option named Color or Cor to render color dots on cards/PDP and sticky bar.
  • Assign a featured image per color variant for correct image swaps.
  • If you use Shopify native color swatches, set swatch values for each color in Admin.

4) Size chart tag (drives the in-page guide)

The size chart shown on the PDP is picked by product tag. Add one of the tags below:

5) Collections (for “You may also like” grid)

  • The related grid uses the first collection the product belongs to.
  • Ensure each product is assigned to at least one collection with other items you want to feature.

6) Optional: reviews block

  • If you enable the rating block, it reads product.metafields.reviews.rating and product.metafields.reviews.rating_count.
  • Confirm your reviews app writes to those metafields; otherwise hide/disable the block.

7) Size guide page link

  • The drawer’s “Guia de tamanhos” link points to /pages/guia-de-tamanhos.
  • Keep that page published and updated.

8) Quick publish checklist

Template Set template to product.new-product.

Sizes Size/Tamanho option exists; values entered; inventory tracked.

Colors Color/Cor option exists; variant images assigned.

Size chart tag One of the required tags added (see table).

Collections At least one collection chosen for related grid.

Reviews Rating metafields available or block disabled.

Guide page /pages/guia-de-tamanhos published.
